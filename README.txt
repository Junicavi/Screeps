Este documento es una breve serie de instrucciones para la correcta utilización de los
scripts creados para el juego Screeps.

----------------------------------------------------------------------------------------
					INSTRUCCIONES
----------------------------------------------------------------------------------------

1.El Spawn debe ser colocado manualmente al inicio de la verificación.
2.Al Spawn se le debe dejar el nombre por default (Spawn1).
3.Se suministrán los scripts en formato .JS y también en .TXT
  Estos últimos se pueden copiar tal cual se presentan para mantener el orden del código
  dentro de los módulos.
4.En caso de ser necesaria la creación de los módulos, se deben mantener los nombres:
	*main
	*role.builder
	*role.harvester
	*role.upgrader
